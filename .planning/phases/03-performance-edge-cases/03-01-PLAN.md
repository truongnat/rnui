## Task 1 — `useCarousel`: width runtime + guard `n === 0`

<action>
Trong `packages/headless/src/hooks/useCarousel.ts`:

1. Xóa `const { width: SCREEN_WIDTH } = Dimensions.get("window")` ở top-level.

2. Gọi **`useWindowDimensions()`** từ `react-native`; đặt `const defaultWidth = Math.max(1, width)` (hoặc `width || 375` — chọn một và ghi trong code) để tránh `itemStep === 0`.

3. Tham số `itemWidth` trong options: nếu `undefined`, dùng `defaultWidth`; nếu có số, dùng số đó.

4. **Guard `n === 0`:**

5. Giữ hành vi **`loop && n === 1`:** `displayData` = `data` (không clone); comment JSDoc trên `displayData` hoặc trên hook.

6. Export type `UseCarouselOptions` cập nhật nếu `itemWidth` trở thành optional rõ hơn — không phá consumer: Carousel vẫn truyền số sau Task 2.

## Task 3 — `Carousel.tsx`: empty UI, `keyExtractor`, width, keys

<action>
Trong `packages/ui/src/components/Carousel/Carousel.tsx`:

1. Xóa module-level `Dimensions.get("window")`.

2. Dùng **`useWindowDimensions()`**; tính `resolvedItemWidth = itemWidth ?? width` (và `Math.max(1, width)` nếu cần đồng bộ Task 1).

3. Khi **`data.length === 0`:** return **`null`** hoặc `<View style={{ height: 0 }} />` — không gọi `renderItem` với phần tử nào; không crash `useCarousel` (đảm bảo hook nhận `data=[]` an toàn sau Task 1).

4. Thêm prop optional **`keyExtractor?: (item: T, index: number) => string`**. Default: `(_, i) => \`carousel-\${i}\``hoặc`String(i)`— **document** trong JSDoc: với object có`id`nên truyền`keyExtractor`.

5. Thay `key={index}` bằng key ổn định:
   - Non-loop: `keyExtractor(item, index)` (index là index trong `displayData` / `data` tùy map — thống nhất: dùng **index trong `displayData`** + map sang `renderItem` index hiện tại).
   - Loop: với clone đầu/cuối, append suffix **`-loop-prev`** / **`-loop-next`** (hoặc tên tương đương trong CONTEXT) để key **unique**.

6. `contentContainerStyle.paddingHorizontal`: dùng `width` từ `useWindowDimensions`, không hằng module.

7. Cập nhật `CarouselProps` export; ghi **changelog / comment** semver minor nếu thêm prop công khai.
   </action>
