# Accordion

Expandable content panels.

## Usage

```tsx
import { Accordion, AccordionSummary, AccordionDetails } from "@rnui/ui";

<Accordion>
  <AccordionSummary>Details</AccordionSummary>
  <AccordionDetails>More content</AccordionDetails>
</Accordion>
```

## Notes

- Refer to the component props in the API reference or source for advanced configuration.
