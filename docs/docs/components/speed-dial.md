# SpeedDial

Floating quick actions menu.

## Usage

```tsx
import { SpeedDial, SpeedDialAction, Icon } from "@truongdq01/ui";

<SpeedDial ariaLabel="Actions" icon={<Icon>+</Icon>}>
  <SpeedDialAction tooltipTitle="Share" />
  <SpeedDialAction tooltipTitle="Save" />
</SpeedDial>
```
