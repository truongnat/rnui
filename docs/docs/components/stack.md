# Stack

Vertical or horizontal layout with consistent spacing.

## Usage

```tsx
import { Stack, Box } from "@rnui/ui";

<Stack spacing="md">
  <Box style={{ height: 32, backgroundColor: "#e2e8f0" }} />
  <Box style={{ height: 32, backgroundColor: "#cbd5f5" }} />
</Stack>
```
