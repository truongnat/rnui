# Breadcrumbs

Navigation breadcrumbs.

## Usage

```tsx
import { Breadcrumbs, Link, Typography } from "@rnui/ui";

<Breadcrumbs>
  <Link href="https://example.com">Home</Link>
  <Link href="https://example.com">Library</Link>
  <Typography>Data</Typography>
</Breadcrumbs>
```
