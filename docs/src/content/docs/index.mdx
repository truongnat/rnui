## Why RNUI?

RNUI is a modern React Native component library designed to help you build beautiful, accessible, and performant mobile applications. Built with TypeScript, powered by Reanimated, and styled with a comprehensive theming system.

<span class="feature-pill">Optimized</span>
<span class="feature-pill">Tree-shakeable</span>

<span class="feature-pill">Modern</span>
<span class="feature-pill">Customizable</span>

<span class="feature-pill">WCAG</span>
<span class="feature-pill">Screen Reader</span>
<span class="feature-pill">Keyboard</span>

## Key Features

### Component Library

<span class="feature-pill">62 Components</span>
<span class="feature-pill">Consistent API</span>
<span class="feature-pill">Responsive</span>

A comprehensive set of UI components covering all common use cases. All components follow the same design patterns and conventions for a consistent experience. Components adapt seamlessly to different screen sizes.

### Theming System

<span class="feature-pill">Multiple Themes</span>
<span class="feature-pill">Custom Tokens</span>
<span class="feature-pill">Dynamic</span>

Pre-built themes including light, dark, and brand-specific options. Define your own design tokens for complete control over the look and feel. Smooth transitions between themes for a polished user experience.

### Form Components

<span class="feature-pill">Validation</span>
<span class="feature-pill">ARIA Labels</span>
<span class="feature-pill">RTL Support</span>

Built-in form validation with error handling to ensure data integrity. Proper ARIA labels and keyboard navigation for accessibility. Full support for RTL languages to reach a global audience.
