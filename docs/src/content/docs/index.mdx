## Why RNUI?

RNUI is a modern React Native component library designed to help you build beautiful, accessible, and performant mobile applications with ease. Built with TypeScript, powered by Reanimated, and styled with a comprehensive theming system.

### 🚀 Built for Performance

- **Optimized Rendering**: Components are built with performance in mind, using React Native Reanimated for smooth animations
- **Tree-shakeable**: Import only what you need to keep your bundle size small
- **TypeScript**: Full type safety for a better development experience

### 🎨 Beautiful by Default

- **Modern Design**: Clean, contemporary aesthetics that look great out of the box
- **Customizable Theming**: Easily adapt the design to match your brand
- **Dark Mode Support**: Built-in dark mode with smooth transitions

### ♿ Accessible Components

- **WCAG Compliant**: Components follow accessibility best practices
- **Screen Reader Support**: Full support for VoiceOver and TalkBack
- **Keyboard Navigation**: Proper keyboard navigation patterns

### 🔧 Developer Experience

- **Headless Hooks**: Build custom UIs with our headless React hooks
- **Comprehensive Documentation**: Clear examples and guides for every component
- **Storybook Integration**: Visual component playground for easy exploration

## Key Features

### Component Library

- **50+ Components**: A comprehensive set of UI components covering all common use cases
- **Consistent API**: All components follow the same design patterns and conventions
- **Responsive Design**: Components adapt seamlessly to different screen sizes

### Theming System

- **Multiple Themes**: Pre-built themes including light, dark, and brand-specific options
- **Custom Tokens**: Define your own design tokens for complete control
- **Dynamic Switching**: Smooth transitions between themes

### Form Components

- **Validation**: Built-in form validation with error handling
- **Accessibility**: Proper ARIA labels and keyboard navigation
- **Internationalization**: Support for RTL languages
