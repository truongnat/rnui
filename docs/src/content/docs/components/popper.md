# Popper

Low level positioning primitive.

## Usage

```tsx
import { Popper } from '@truongdq01/ui';

<Popper open anchorEl={{ x: 100, y: 120, width: 40, height: 20 }}>
  Content
</Popper>;
```

- Refer to the component props in the API reference or source for advanced configuration.
