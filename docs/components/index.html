UI Components

A comprehensive collection of 62+ React Native UI components.

📄️ Typography

Typography component for consistent text styling with predefined variants.

📄️ Accordion

Accordions display content in expandable/collapsible sections, perfect for FAQs and grouped content.

📄️ Box

A versatile container component with MUI-style sx prop for rapid styling.

📄️ Stack

A layout component for arranging items vertically or horizontally with consistent spacing.

📄️ Chip

Chips are compact elements that represent an input, attribute, or action. They can display icons, avatars, and delete actions.

📄️ Image

The Image component provides a performant image loading experience with built-in placeholder effects and smooth fade-in animations powered by Reanimated 3.

📄️ Icon

Standardized icon component using Lucide React Native with 70+ icons available.

📄️ Modal

Modals are overlay containers for focused tasks and important content that requires user attention.

📄️ Drawer

Drawers are side panels that slide in from the screen edges, commonly used for navigation menus.

📄️ Tooltip

Tooltips display informative text when users interact with the trigger element. They appear on hover or long-press.

📄️ AnimatedList

A performant animated list component built on top of FlashList with enter/exit animations powered by Reanimated 3.

📄️ Carousel

A swipeable carousel component with gesture support, auto-play, and customizable pagination indicators.

📄️ DatePicker

A fully customizable date picker with calendar view, range selection, and theme support.

📄️ SegmentedControl

A horizontal segmented control for mutually exclusive option selection. Commonly used for tabs, filters, or view toggles.

📄️ Alert

Alerts provide contextual feedback messages for typical user actions.

📄️ AppBar

Top application bar.

📄️ Autocomplete

Searchable select with suggestion list.

📄️ Avatar

The Avatar component displays a user image, initials, or a fallback icon. Use AvatarGroup to show multiple users with overlap.

📄️ Badge

Compact status label with variants and an optional icon.

📄️ BottomNavigation

Bottom navigation actions.

📄️ BottomSheet

Gesture-driven sheet with multi snap points, velocity-based dismiss, and an imperative ref API. All gesture and animation logic runs on the UI thread.

📄️ Breadcrumbs

Navigation breadcrumbs.

📄️ ButtonGroup

Group related buttons together.

📄️ Button

The Button component wraps usePressable from the headless layer. All animation runs on the UI thread via Reanimated 3.

📄️ Card

Cards are surface containers with consistent padding, radius, and optional press feedback.

📄️ Checkbox

Checkboxes support controlled or uncontrolled state, indeterminate state, and optional label/description.

📄️ CircularProgress

Circular loading indicator.

📄️ Dialog

Modal dialog surface.

📄️ Divider

Dividers separate content vertically or horizontally, with optional label text.

📄️ EmptyState

Empty states provide a consistent layout for blank screens or empty lists.

📄️ Fab

Floating action button for primary actions.

📄️ FormControl

Context wrapper for form inputs.

📄️ FormField

FormField wraps inputs with a consistent label, error, and helper text layout. FormGroup stacks fields with uniform spacing.

📄️ Grid

Responsive layout grid inspired by MUI.

📄️ ImageList

Responsive image grid.

📄️ Input

The Input component provides a styled text input with support for labels, error messages, helper text, and icons.

📄️ LinearProgress

Horizontal progress bar.

📄️ Link

Inline link that opens external URLs.

📄️ List

High-performance list built on FlashList, with optional swipe actions and section headers.

📄️ Menu

Popup menu list.

📄️ OTP Input

A specialized input for One-Time Passwords with automatic focus management and numeric filtering.

📄️ Pagination

Page navigation controls.

📄️ Paper

Surface container with elevation or outline.

📄️ PasswordInput

The PasswordInput component extends the base Input and includes a built-in visibility toggle.

📄️ Popover

Floating overlay anchored to a position.

📄️ Popper

Low level positioning primitive.

📄️ Pressable

Pressable wraps the headless usePressable hook with gesture-driven feedback on the UI thread.

📄️ RadioGroup

RadioGroup renders a list of options using the RadioItem component with shared selection state.

📄️ Rating

Star rating input.

📄️ Select

Bottom-sheet select with optional search and multi-select.

📄️ Skeleton

Skeletons provide loading placeholders with optional shimmer.

📄️ Slider

A gesture-driven slider with optional value display, range labels, and step marks.

📄️ Snackbar

Transient feedback at bottom of screen.

📄️ SpeedDial

Floating quick actions menu.

📄️ Stepper

Progress through steps.

📄️ Switch

Switches toggle a boolean state and support optional label/description.

📄️ Table

A flexible data table component with optional headless logic for sorting, pagination, and selection.

📄️ Tabs

Switch between sections.

📄️ TextArea

Multi-line text input with label, helper text, error state, and auto-resizing.

📄️ TextField

Full featured input with label and helper text.

📄️ Timeline

Chronological event list.

📄️ Toast

Transient notifications powered by useToast and rendered by ToastContainer.

📄️ ToggleButton

Exclusive or multi select toggle group.
