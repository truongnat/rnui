Architecture

Two ways to build

Choose the approach that fits your needs

Styled Components

Pre-built, beautifully styled components ready to use out of the box.

import { Button } from "@rnui/ui"; <Button variant="primary" />
  • Quick setup
  • Consistent design
  • Theme support

Headless Hooks

Logic and state management with complete styling freedom.

import { usePressable } from "@rnui/headless"; const { gesture, style } = usePressable();
  • Full control
  • Custom designs
  • Accessibility built-in
